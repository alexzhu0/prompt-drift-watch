# prompt-drift-watch v0.2.1

Trend-aligned release for coding-agent instruction review.

## Added

- AGENTS.md-style example pair for baseline and changed agent instructions.
- Risk category reporting for approval, citation, safety, scope, and verification.
- GitHub Actions annotation output through `--format github`.
- Expanded default terms for workspace, sandbox, destructive commands, and vague agent behavior.

## Try It

```bash
PYTHONPATH=src python3 -m prompt_drift_watch examples/baseline_agents.md examples/changed_agents.md --show-diff
PYTHONPATH=src python3 -m prompt_drift_watch examples/baseline_agents.md examples/changed_agents.md --format github --fail-on-risk 80
```
