# v0.1.0

## What changed

- Initial focused release of `prompt-drift-watch`.
- Detects removed guardrails and vague prompt additions.
- Supports text and JSON output.
- Supports CI-style failure with `--fail-on-risk`.

## Try it

```bash
PYTHONPATH=src python3 -m prompt_drift_watch examples/baseline_prompt.md examples/changed_prompt.md
```

## Notes

- This release is deterministic and standard-library only.
- Please open issues with concrete prompt diffs and expected output.
