# prompt-drift-watch v0.2.0

## What Changed

- Added custom rule packs with `--rules`.
- Added compact diff snippets with `--show-diff`.
- Included rule-pack metadata in JSON reports.

## Quickstart

```bash
PYTHONPATH=src python3 -m prompt_drift_watch examples/baseline_prompt.md examples/changed_prompt.md --show-diff
```

PYTHONPATH=src python3 -m unittest discover -s tests
